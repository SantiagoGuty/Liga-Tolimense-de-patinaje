# Liga Tolimense de Patinaje — Web App

Official website for **La Liga Tolimense de Patinaje** built with **React + Vite + TypeScript**, deployed on **AWS Amplify**.

Status: Active development (frontend live, backend features in progress)

## Features

- Responsive website for desktop and mobile
- News pages and detail views
- Courses and modalities pages
- Guides and resolutions sections
- Reusable banners and shared UI sections
- AWS Amplify auth integration

---

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Routing:** React Router
- **Styling:** CSS
- **Auth / Cloud (WIP):** AWS Amplify + Cognito
- **Storage (WIP):** AWS S3
- **Deployment:** AWS Amplify Hosting
